Quem somos nós?



A SmartMovement é sua primeira opção quando se trata de operação e análise de fluxo de pessoas, tendo como foco principal objetivo proporcionar a validação e checagem da informação de fluxo de pessoas de forma segura, permitindo o uso desses dados para tomada de decisões estratégicas.

Nossa missão

Fornecer dados estratégicos para melhorar o desempenho de shoppings com uso de tecnologia.
